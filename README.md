#FAQ accordion for Q&A with React

## Description
this FAQ uses in Q&A part of website.
if you click on question then answer of that question will move down with animation
this project created with react , bootstrap , sass

## Installation
### 3 step

#### 1.push Repo
#### 2.npm install
#### 3.npm start
